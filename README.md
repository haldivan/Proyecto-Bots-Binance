# Proyecto Bots Binance

Sistema modular para el desarrollo, validación y optimización de estrategias de trading automatizado sobre Binance.

## Componentes

- Bot Crypto
- Bot TradFi
- Analizador Estadístico
- Motor Estadístico
- Diagnóstico Automático
- Motor de Evidencia
- Motor de Hallazgos Estratégicos
